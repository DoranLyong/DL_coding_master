## Required Knowledge 
### (1) Arithmetic operation
* 가감승제 
```python 
>>> 1 - 2 
-1 

>> 4 * 5 
20 

>> 7 / 5 
python3
1.4

python2
1

>> 3 ** 2 
9
```
※ 나눗셈 연산 
> python2 : 정수끼리 계산한 결과는 반드시 정수, < int ><br/>

> python3 : 정수를 나눈 결과는 실수(부동소수점), < float ><br/>

<br/>

### (2) Data type 
* 파이썬의 자료형은 총 4 개 
> < bool >, < int >, < float >, < str >
```python 
>>> type(10)
<class 'int'>

>>> type("hello")
<class 'str'>
```
※ Python은 모든 것을 객체<sub>object</sub> 개념으로 봄
